# Support

## Community support

- Use GitHub Issues for bug reports and feature requests.
- Use Discussions (once enabled) for design questions and implementation help.

## Commercial/support inquiries

- legal@velariq.com

## What to include in bug reports

- component/path affected,
- reproduction steps,
- expected vs actual behavior,
- environment details (OS, Python version, relevant deps),
- logs/error output (with secrets redacted).

## Security issues

Use private channels only. See `SECURITY.md`.
