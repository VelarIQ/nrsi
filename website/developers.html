Builder onboarding

Start building with NRSI in minutes

Clone the repo, install dependencies, and run runtime/tooling tests. Move to enterprise support only when you need production guarantees.

1) Clone and install

git clone https://github.com/VelarIQ/nrsi.git
cd nrsi
python -m pip install -r global-rollout/requirements.txt

2) Run baseline tests

python -m pytest global-rollout/tests/test_nrsi_lang.py -v
python -m pytest global-rollout/tests -v

Core Paths

  • `global-rollout/nrsi/` runtime + language
  • `global-rollout/tools/nrsi-lang/` toolchain
  • `global-rollout/tests/` validation suites

Contribute

  • Read CONTRIBUTING and SECURITY policies
  • Open issues and small focused PRs
  • Include test evidence with each change
